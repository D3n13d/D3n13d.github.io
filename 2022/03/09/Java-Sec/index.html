Java 安全学习

写在前面，这是一个成长文，记录笔者学习Java安全的过程。

Java相较于其他语言的优势，编写完程序之后，可以将其编译为可执行模式，而这种优势由JVM提供技术支持

JDK：JRE的超集，包含编译器和调试器等用于程序开发的文件 JRE

JRE：Java虚拟机、库函数、运行Java应用程序和Applet所必须文件

所以运行java程序需要JRE，而开发Java程序则需要JDK

JVM：一个虚拟的用于执行bytecodes 字节码的计算机。它也定义了指令集、寄存器集、结构栈、垃圾收集堆、内存区域。

.java -> .class -> .exe
